스마틱스 UI 도우미

호스트 프로젝트를 수정하지 않고 로컬 웹 페이지에 agent-ui-annotation 툴바를 주입할 수 있는 내부 전용 독립 도구입니다.

  1. 이 도구를 로컬에서 실행합니다.
    2. [npm i 실행 후 npm run dev 실행]
  2. 아래 북마클릿복사 또는 Inject URL을 복사합니다.
  3. 사용하고 싶은 싶은 로컬 웹 페이지를 엽니다.
  4. 개발자도구(F12)를 열고 북마클릿에서 복사된 코드를 실행합니다.
북마클릿

CSP가 강한 페이지는 외부 module script 주입을 차단할 수 있습니다. 이 경우 해당 페이지에서는 툴바가 로드되지 않을 수 있습니다.
